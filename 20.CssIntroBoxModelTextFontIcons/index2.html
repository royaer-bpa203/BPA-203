CSS Background Özellikleri

HTML elementlerinin arka plan efektlerini tanımlama

Background Özelliği Nedir?

Background özelliği, HTML'deki belirli bir elementin arka plan efektlerini tanımlamak için kullanılır.

Bu bir CSS kısaltma özelliğidir çünkü arka plan resmi ekleyebilir, belirli bir resmi tekrarlayabilir ve CSS oluşturmak için Background özelliklerini kullanabiliriz.

Örnek Kullanım:

.element {
  background: #f0f0f0 url('resim.jpg') no-repeat center center;
}

Background Özellikleri

Background özelliği, HTML'deki belirli bir elementin arka plan efektlerini tanımlamak için kullanılır.

Bu bir CSS kısaltma özelliğidir çünkü harfler ekleyebilir, belirli bir resim örneğini tekrarlayabilir ve CSS oluşturmak için kullanabiliriz.

Background Bileşenleri:

  • background-color
  • background-image
  • background-repeat
  • background-position
  • background-size
  • background-attachment
